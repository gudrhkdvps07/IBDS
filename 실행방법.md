### 실행하기 전 설정
1. docker 먼저 다운
2. 관리자 전용 터미널에서 dvwa 등록
    docker run -d --name dvwa -p 8080:80 vulnerables/web-dvwa
3. docker start dvwa

### 실행
1. mitm proxy 실행
    mitmdump -s capture.py -p 8081 
2. 크롤러 실행
    $env:PROXY_URL       = "http://127.0.0.1:8081"
    $env:TARGET_URL      = "http://localhost:8080"
    $env:DEMO_DVWA_AUTH  = "1"
    $env:DEMO_DVWA_PHPSESSID = "실제세션값"

    python -m crawl.crawler
