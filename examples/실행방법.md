## 사전 설치
- Docker Desktop 설치 및 실행

---

## 1. DVWA 설정 (테스트 대상 서버)

관리자 터미널에서 실행:
```powershell
docker run -d --name dvwa -p 8080:80 vulnerables/web-dvwa
```

이미 등록된 경우:
```powershell
docker start dvwa
```

브라우저에서 `http://localhost:8080` 접속 → Setup 페이지에서 DB 초기화 → admin/password 로그인

---

## 2. ZAP 설정 (프록시 / 수집)

### 최초 1회만 실행
```powershell
.\zap_setup.ps1
```
자동으로 다음을 처리함:
- ZAP Docker 이미지 pull
- ZAP 컨테이너 시작 (포트 8081)
- ZAP 초기화 완료 대기
- `zap_root_ca.cer` 인증서를 프로젝트 루트에 저장

### 이후 실행 시
```powershell
docker start zap
```

### ZAP 종료
```powershell
docker stop zap
```

---

## 3. Firefox 설정 (최초 1회)

### 프록시 설정
`about:preferences#general` → 네트워크 설정 → 수동 프록시 설정
- HTTP 프록시: `127.0.0.1` / 포트: `8081`
- HTTPS도 동일하게 적용 체크

### 인증서 설치
`about:preferences#privacy` → Certificates → View Certificates → Authorities → Import
- `zap_root_ca.cer` 선택
- "Trust this CA to identify websites" 체크 → OK

---

## 4. 수집 실행

Firefox로 `http://localhost:8080` 탐색 (로그인 포함, 여러 페이지 이동)

```powershell
python main_collect.py
```

결과 확인:
- `results/collection/raw_zap_messages_sample.json` — ZAP raw 메시지 샘플
- `results/collection/collected_targets.json` — 변환된 RequestTarget 목록
